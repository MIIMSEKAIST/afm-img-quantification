# Precomputed results

These CSVs are the reference outputs reported in the paper. Re-running the
scripts on the ground-truth data (see `../data/README.md`) reproduces them; the
3 auxiliary IBW files in the Zenodo deposit are ignored via
`../data/benchmark_scans.txt`, so the counts (26 scans → 104 cases) are stable.

| file | rows | what | regenerate |
|------|------|------|------------|
| `benchmark104.csv` | 104 | per-case validation metrics (case_id, scan, material, colormap, nMAE_pct, mean_error_pct, std_error_pct, normalized_EMD_pct, SSIM, valid_pixel_ratio, mean_cielab_distance) — the data behind Fig. 3 and Supplementary Data 1 | `python benchmark/reproduce_benchmark.py --ibw-dir data/esm_ibw --out results/benchmark104.csv` |
| `robustness_sweep.csv` | 1040 | JPEG-quality and figure-resolution sweep (sweep, colormap, param, nMAE, SSIM); bilinear downsampling — Supplementary Note 3 / Fig. S4 | `python benchmark/robustness_sweep.py --ibw-dir data/esm_ibw --out results/robustness_sweep.csv` |
| `meta_scanlevel.csv` | 104 | scan-level median amplitudes (scan, material, colormap, gt_med, rec_med) consumed by the meta-analysis — Fig. 5 / Table 1 | regenerated by `python figures/make_figure_parts.py` (parts_fig5) if absent |

Headline reproduction: `reproduce_benchmark.py` → nMAE 2.09 ± 0.76 %, SSIM
0.807 ± 0.105 (copper 1.74 / hot 1.83 / jet 2.13 / viridis 2.67);
`diffusivity_consistency.py` → scan-median deviation 0.90 ± 0.77 %, r(logD) 0.91;
`analysis/meta_analysis.py` → ranking LMNO > LICGC > NCM811 preserved across all
corpora.
